Lolicon API

Lolicon API 官网

调用方法官网有，这里只复述主要部分。

调用方法：

GET https://api.lolicon.app/setu/v2
Content-Type: application/json

loli.st API

警告：这API不稳定，已损坏50%
故没有官网，未知。

调用方法：

####接口1：

https://api.loli.st/pixiv/?mode=

mode 备注
daily(_r18) 日榜(r18)
weekily(_r18) 周榜(r18)
monthly() 月榜

####接口2：

https://api.loli.st/pixiv/random.php?

type orther
json 返回json值
none 网站反显
r18 orther
true 随机返回一张图(含R18)
flase 随机返回一张图

pixiv.cat

pixiv 反向代理

使用方式
單張作品
適用於一個作品ID中只有一張圖片的作品

https://pixiv.cat/Pixiv作品數字ID.jpg/png/gif

多張作品 (漫畫模式)
適用於一個作品ID中有多張圖片的作品

https://pixiv.cat/Pixiv作品數字ID-第幾張圖.jpg/png/gif

備用鏡像網域: pixiv.re

圖片為動態產生，網址結尾副檔名部分無實際用途，準確檔案類型會以 Content-Type header 發送。

随机二次元图片API-樱花

随机二次元图片API-樱花 官网

调用方法：

https://www.dmoe.cc/random.php

参数：

type orther
json 返回json格式
none 直接返回图片

img 例子：https://www.dmoe.cc/random.php

随机图片API

随机图片API 官网

调用方法
写真调用地址

https://api.pixivweb.com/api.php?

二次元调用地址

https://api.pixivweb.com/anime18r.php?

黑白艺术（18R）

https://api.pixivweb.com/bw.php?

return Column2
json 返回json格式
https 图片
img 图片

img(18r)例子：https://api.pixivweb.com/anime18r.php?return=img
